1️⃣ What is the difference between var, let, and const?
Answer:var is older,function-scoped and can cause bugs due to hoisting.
let is block-scoped, safer for variables that will change.
const is for constants, values cannot be reassigned.

2️⃣ What is the spread operator (...)?
Answer:Expands an iterable (like an array or object) into individual elements.

3️⃣ What is the difference between map(), filter(), and forEach()?
Answer:map():Transforms each element
filter():Filters elements based on a condition
forEach():Iterates over array for side-effects (does not return array)

4️⃣ What is an arrow function?
Answer:Shorter syntax for writing functions and does not have its own this.

5️⃣ What are template literals?
Answer:Enclosed in backticks ` instead of quotes.It can include expressions using ${} and supports multi-line strings.
